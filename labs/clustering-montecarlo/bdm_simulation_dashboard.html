⚙️ 参数设定注解

AR(1) ρ：自相关强度。值越大，残差随时间累积的“粘性”和依赖度越强，是导致OLS标准误偏小的根源。支持输入负值观察过度保守现象。

模拟次数：平行宇宙的数量。高数值（如500次以上）将提供极为平滑精确的核密度与拒绝率估计，低数值则能瞬间渲染完成。

省份组数 (G)：即聚类组数。渐近聚类要求 G → ∞。在此输入极小值（如 G=3）可直观见证小样本偏误下聚类标准误如何也发生崩塌。

每组观测数量 (N)：每个省份内部微观观测体的样本数量。

📚 为什么我们需要对标准误“体检”？

“我们应该在多大程度上信任双重差分估计？”

—— Bertrand, Duflo, and Mullainathan (2004), QJE

在实证经济学中，双重差分法（DID）几乎是评估政策效应的黄金标准。 然而，现实世界中的大多数宏观或区域变量（如各省 GDP、工资、税率）都具有极强的路径依赖/时间自相关性（即今年的状态深受昨天状态的影响）。

εit = ρ εi,t-1 + νit

🤯 自相关如何欺骗了传统统计推断？

传统的经典标准误（IID）假设所有的观测值残差之间是彻底独立的。 然而，在 AR(1) 过程自相关系数 ρ > 0 时，同一个省份不同年份的残差里其实包含着高度重叠的“本地冲击信号”。

如果使用经典算法，相当于强行把这些重叠信息判定为独立新样本。这会使得估计出来的标准误差小得可怜，进而让算出来的 t-统计量异常膨胀。 BDM (2004) 用蒙特卡洛实验首次向世界敲响警钟：即使政策完全没有半点实际效果，传统方法依然有接近 45% 的概率宣告政策显著！

🛠️ 标准误家族的“对角线革命”

为了纠正这一灾难，我们需要调整残差的方差-协方差矩阵（VCE）：

标准误类型 物理假设直觉 对自相关的防范效果
IID 经典标准误 假设每个观测样本都是完全孤立的岛屿，毫无关联。 🚫 完全无效。在自相关下假阳性率飙升至 45%。
HC1 稳健标准误 允许不同个体波动大，但仍然坚信时序之间完全独立。 🚫 完全无效。仅纠正异方差，无视自相关，依然崩溃。
单向聚类标准误 释放省份内部所有年份的限制，允许任意自相关与异方差。 ✅ 极其优秀。样本组数足够时，稳妥收敛于名义 5% 线。

clear all
set obs 500         // 500 个观测体
gen id = _n
expand 10           // 10 个时期
bysort id: gen t = _n

* 构造自相关残差 (自相关系数 rho = 0.8)
gen eps = .
bysort id: replace eps = rnormal(0, 1) if t == 1
bysort id: replace eps = 0.8 * eps[_n-1] + rnormal(0, 1) if t > 1

* 设定省份级聚类层级 (50 个组)
gen g = ceil(id / 10)
save "simdata.dta", replace

* 运行蒙特卡洛推演
forv i = 1/300 {
    use "simdata.dta", clear
    * 随机分配 20% 省份为政策试点组 (治疗组)
    qui g grnum = runiform(0,1) if t == 1
    bysort g: replace grnum = grnum[1]
    qui gen year = (t >= 5)
    qui gen treat = (grnum <= 0.2) * year // 真实效应为 0
    qui gen y = treat * 0 + eps
    
    * 经典OLS (不聚类)
    qui reg y treat
    * 组内聚类标准误
    qui reg y treat, vce(cluster g)
}
🐍 Python numpy/scipy 极速仿真引擎
# 双向去均值与 OLS 聚类核心计算逻辑
import numpy as np
from scipy import stats

def run_sim(rho=0.8, n_units=500, n_periods=10, n_groups=50):
    # 1. 构造 AR(1) 误差
    eps = np.zeros((n_units, n_periods))
    eps[:, 0] = np.random.normal(0, 1, n_units)
    for t in range(1, n_periods):
        eps[:, t] = rho * eps[:, t - 1] + np.random.normal(0, 1, n_units)
        
    # 2. 随机分配治疗组，设定真实效应为 0
    D = np.zeros((n_units, n_periods))
    treated = np.random.choice(n_groups, size=n_groups//5, replace=False)
    group_ids = np.repeat(np.arange(n_groups), n_units // n_groups)
    D[np.isin(group_ids, treated), 5:] = 1
    Y = 0 * D + eps
    
    # 3. 面板双向去均值 (Within Transformation)
    Y_dm = Y - Y.mean(axis=1, keepdims=True) - Y.mean(axis=0) + Y.mean()
    D_dm = D - D.mean(axis=1, keepdims=True) - D.mean(axis=0) + D.mean()
    
    # 4. 估计系数与单向聚类标准误
    beta_hat = (D_dm * Y_dm).sum() / (D_dm ** 2).sum()
    resid = Y_dm - beta_hat * D_dm
    group_scores = (D_dm.reshape(n_groups, -1) * resid.reshape(n_groups, -1)).sum(axis=1)
    se_cl = np.sqrt((group_scores ** 2).sum() / (D_dm ** 2).sum() ** 2 * n_groups / (n_groups - 1))
    return beta_hat, se_cl

🧪 蒙特卡洛平行宇宙实验设定与思路

实验设定：本实验室旨在通过计算机在几毫秒内推演出多达数百个“平行宇宙（即 Monte Carlo 模拟迭代）”。在每个宇宙的数据生成过程 (DGP) 中，我们人为且明确地定死真实政策效应等于 0（即政策是一剂完全无效的生理盐水，零假设 H0: β = 0 绝对成立）。

实验思路：当研究者设定显著性水平为 5% 时，意味着他们允许 5% 的统计推断出现假阳性（误诊）。如果一个标准误估计方法是正确且诚实的，那么在数百个宇宙中，它应该有且仅有大约 5% 的比例错误地拒绝零假设（宣称政策有用）。通过拖动左侧面板的 AR(1) 自相关系数 ρ，我们人为制造时序残留记忆的强弱，直观检验经典OLS (IID)、稳健标准误 (HC1) 以及省份级聚类标准误 (Cluster) 对这 5% 显著水平的防守能力，亲眼见证 BDM (2004) 中 45% 过度拒绝的震撼效应。

🧬 点击展开：数据生成与构造过程 (DGP) 技术细节

本实验的数据生成过程 (DGP) 严格参照 BDM (2004) 及学术规范，按如下五个步骤在内存中动态合成：

  1. 面板维度构建： 定义一个具有 G 个省份、每省 N 个企业单位、观测期固定为 T = 10 期的平衡面板。总观测个体量为 I = G × N，总样本量为 I × T。
  2. AR(1) 残差时序依赖生成： 对于每个独立的企业个体 i，其 10 期的残差序列 εit 基于一阶自相关过程递推：
    εi1 ∼ N(0, 1)
    εit = ρεi, t-1 + νit   (其中 νit ∼ N(0, 1))
    这里的 ρ 即为左侧拖动的自相关系数。当 ρ > 0 时，本期残差中会继承前期的冲击，制造路径依赖。
  3. 政策虚拟变量 Dit 构造： 随机抽取正好 20% 的省份 作为政策试点地区（治疗组）。政策时点定在 t ≥ 5。因此，若企业 i 所在省份被抽中且处于 t ≥ 5，则 Dit = 1，否则 Dit = 0。
  4. 观测因变量 Yit 合成： 基于结构方程合成观测值：
    Yit = ui + βDit + εit
    横截面异质冲击 ui ∼ N(0, 1) 代表企业个体固定效应。由于我们测试的是假阳性率（无中生有的虚假显著性），我们必须将真实的政策效应 β 定死为 0，故实际生成公式为 Yit = ui + 0 × Dit + εit。
  5. 去双向固定效应估计： 对数据进行双向去均值（Within-panel Transformation），消除个体固定效应 ui 与时间趋势，然后基于普通最小二乘法 (OLS) 估计政策系数 β̂ 并计算对应标准误，从而完成一次平行宇宙的推演。

🚀 实证前沿：当常规聚类标准误也失效时

虽然聚类标准误（Clustered SEs）完美攻克了面板序列自相关问题，但计量经济学前沿研究发现，在一些极端边界条件下，常规聚类标准误同样会面临推断失效：

边界突破 1：小样本群组 (G < 30)

为什么失效？

聚类标准误的有效性依赖于“渐近大样本理论”（要求组数 G → ∞）。当聚类组数太少（如仅有 10 个省份）时，CRVE 对残差方差的估计存在向下的偏差，导致假阳性率再次抬头。

前沿对策：Wild Cluster Bootstrap

利用当前估计的残差，通过“抛硬币（拉德马赫权重）”人工构造数千个高质量的“仿生平行宇宙”，在小样本甚至只有一个治疗组省份的情况下，依然能实现完美的 5% 拒绝率。

边界突破 2：多维重叠关联

为什么失效？

很多时候残差不仅在省份（个体）内部相关，在年份上（例如特定年份爆发金融危机、疫情等系统冲击）也存在交叉相关。如果只按省份进行聚类，依然会低估年份层面的系统共震。

前沿对策：双重聚类 (Two-way Cluster)

由 Cameron, Gelbach, and Miller (2011) 提出，通过同时在省份和年份两个维度对协方差矩阵对角块进行松绑解放：V2way = Vid + Vtime - Vid ∩ time，从时空双维度锁死推断漏洞。
