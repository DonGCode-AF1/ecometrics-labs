Neyman 正交性与 Double Machine Learning

用一个“培训影响工资”的最小模拟说明：为什么普通回归会把背景差异误当成因果效应， 以及 DML 如何通过双重残差化，让第一阶段预测模型的小误差不容易污染最终的处理效应估计。

背景变量 X 同时影响处理 D 和结果 Y；目标是识别 D 对 Y 的因果效应。

核心直觉 混杂问题 DML 步骤 交互模拟 正交性 最小代码
一句话

先清理背景，再估计剩余变动之间的关系

Neyman 正交性的直觉是：让机器学习负责从 Y 和 D 中清理背景变量 X 能解释的部分，但不要让机器学习模型直接给出因果结论。最后只用“清理背景之后剩下的变动” 来估计处理效应。

Y = θD + g(X) + U,     D = m(X) + V

其中 θ 是真正关心的处理效应；g(X) 与 m(X) 是辅助函数，负责吸收背景变量造成的系统性差异。

为什么直接回归会错？

混杂来自同一个背景变量

假设教育、能力、地区等背景变量 X 同时影响两件事：一个人是否参加培训 D，以及他的工资 Y。这时直接比较参加培训者和未参加者， 比较到的不是纯粹培训效应。

直接回归 Y ~ D 会把“背景更好的人更可能参加培训”也算进培训效果里。

普通回归使用原始 D 和原始 Y。斜率会混入 X 的影响，因此通常偏离真实效应。

DML 做了什么？

DML 的重点不是“用了机器学习”本身，而是下面这个正交化结构。

1. 预测结果用 X 预测 Y：
ĝ(X) ≈ E[Y|X]
2. 得到结果残差Ỹ = Y − ĝ(X)
3. 预测处理用 X 预测 D：
m̂(X) ≈ E[D|X]
4. 得到处理残差D̃ = D − m̂(X)
最终估计：   Ỹ = θD̃ + error

换句话说，DML 问的是：在去掉背景变量 X 能解释的部分之后，培训中剩下的变化，是否还能解释工资中剩下的变化？

交互模拟：普通回归 vs 正交残差回归

原始数据中的关系：D 和 Y 的相关性混入了背景变量 X。

残差数据中的关系：D̃ 和 Ỹ 更接近真正的处理效应。

Neyman 正交性体现在哪里？

关键在这个 moment 结构：

ψ(W; θ, η) = (D − m(X)) [Y − g(X) − θ(D − m(X))]

它的设计目标是：第一阶段的 ĝ(X) 和 m̂(X) 出现小误差时，这些误差不会以“一阶”的方式直接传导到 θ。

直白理解：第一阶段模型抖了一点，最终 θ 不会跟着线性大幅抖动；主要影响被压到二阶项，比如误差平方或误差乘积。

所以 DML 不是说第一阶段可以乱估，而是说第一阶段不必完美。只要估得足够合理，正交结构会降低它对最终因果参数的污染。

最小 Python 代码

下面是可复制运行的最小版本。它用随机森林估计两个辅助函数，再用残差回归估计 θ。

import numpy as np
from sklearn.ensemble import RandomForestRegressor
from sklearn.linear_model import LinearRegression
from sklearn.model_selection import train_test_split

np.random.seed(42)
n = 2000
p = 5
X = np.random.normal(size=(n, p))
theta_true = 2.0

# 背景 X 影响培训 D
m_X = 0.8 * X[:, 0] - 0.5 * X[:, 1] + 0.3 * X[:, 2] ** 2
D = m_X + np.random.normal(size=n)

# 背景 X 也影响工资 Y
g_X = 1.5 * X[:, 0] + np.sin(X[:, 1]) + 0.5 * X[:, 2] ** 2
Y = theta_true * D + g_X + np.random.normal(size=n)

# 错误做法：直接回归 Y ~ D
naive = LinearRegression()
naive.fit(D.reshape(-1, 1), Y)
theta_naive = naive.coef_[0]

# 样本切分：避免同一批数据既训练 ML 又做最终估计
X_train, X_test, D_train, D_test, Y_train, Y_test = train_test_split(
    X, D, Y, test_size=0.5, random_state=42
)

model_y = RandomForestRegressor(n_estimators=200, max_depth=5, random_state=42)
model_d = RandomForestRegressor(n_estimators=200, max_depth=5, random_state=42)
model_y.fit(X_train, Y_train)
model_d.fit(X_train, D_train)

Y_res = Y_test - model_y.predict(X_test)
D_res = D_test - model_d.predict(X_test)

dml = LinearRegression(fit_intercept=False)
dml.fit(D_res.reshape(-1, 1), Y_res)
theta_dml = dml.coef_[0]

print("真实 theta:", theta_true)
print("直接回归 theta:", theta_naive)
print("DML / Neyman 正交 theta:", theta_dml)

一句话收束

普通回归问的是：参加培训的人工资是否更高？

DML / Neyman 正交问的是：在扣除背景变量 X 能解释的部分之后，培训的剩余变化是否还能解释工资的剩余变化？

不要让机器学习直接给出因果结论，而是让它清理混杂，再用清理后的剩余变动估计因果效应。
