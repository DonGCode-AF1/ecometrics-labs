DML 交叉拟合：折外残差与因果估计

这个网页用一个 12 个观测值的极小样本解释 cross-fitting：每个样本都由“没见过它的模型”预测， 得到折外残差，再把所有残差合起来估计因果参数 θ。它回答的是：DML 为什么必须先分折、再折外预测？

核心直觉

交叉拟合不是为了炫技，而是为了避免第一阶段机器学习模型“偷看本样本答案”。在 DML 中，每个样本的 Y 残差 和 D 残差 都应该来自折外预测。

Ỹᵢ = Yᵢ − ĝ(−k)(Xᵢ),     D̃ᵢ = Dᵢ − m̂(−k)(Xᵢ),     i ∈ Iₖ

上标 (−k) 的意思是：模型训练时没有使用第 k 折数据；它只是在第 k 折上做预测。

阅读顺序

可以把 DML 拆成两层理解：第一层是 Neyman 正交性，说明为什么“残差对残差”的估计对第一阶段误差不那么敏感； 第二层就是本页的 交叉拟合，说明这些残差为什么要用折外预测生成。

如果模型在同一个样本上训练又预测，残差可能被过拟合压得过小。交叉拟合强制每个样本只接受“外部模型”的预测，从而让第二阶段的 θ̂ 更像真正的样本外因果估计。

2. 三折交叉拟合如何轮流工作？

设 K = 3。每一轮，两个折用于训练一阶段模型，一个折用于生成折外残差。

第 1 轮：生成第 1 折残差
留出/预测：第 1 折 训练：第 2 折 训练：第 3 折

用第 2、3 折训练 ĝ 和 m̂，只在第 1 折上预测。

第 2 轮：生成第 2 折残差
训练：第 1 折 留出/预测：第 2 折 训练：第 3 折

用第 1、3 折训练 ĝ 和 m̂，只在第 2 折上预测。

第 3 轮：生成第 3 折残差
训练：第 1 折 训练：第 2 折 留出/预测：第 3 折

用第 1、2 折训练 ĝ 和 m̂，只在第 3 折上预测。

图 2：橙色表示本轮留出折，蓝色表示训练折。每个样本最终都会获得一次折外预测。

3. 折外残差长什么样？

一阶段模型做两件事：用 X 预测 Y，得到 ĝ(X)；用 X 预测 D，得到 m̂(X)。 对每个留出样本，计算：

Ỹ = Y − ĝ(X),     D̃ = D − m̂(X)

4. 从残差汇总到 θ

所有样本都有折外残差后，把它们合并起来估计：

θ̂ = Σ D̃ᵢỸᵢ / Σ D̃ᵢ²

5. 过程拆解：你应该如何在脑中理解它

1
把样本分成 K 折。 每一折都会轮流当一次“考试集”。考试集不能参与本轮一阶段模型训练。
2
在 K−1 折上训练两个模型。 一个模型学 X 如何预测 Y；另一个模型学 X 如何预测 D。
3
在留出折上生成折外残差。 这些预测是“没见过本样本”的预测，所以较少受样本内过拟合污染。
4
所有折都跑完后，汇总残差。 每个样本都有一对残差：Ỹ 和 D̃。
5
用 Ỹ 对 D̃ 回归。 回归斜率就是 θ̂。它表达的是：去掉背景 X 后，D 的剩余变化对 Y 的剩余变化有什么影响。

6. 最小 Python 样板代码

import numpy as np
from sklearn.model_selection import KFold
from sklearn.ensemble import RandomForestRegressor

# Y: 结果变量；D: 处理变量；X: 控制变量矩阵
K = 3
kf = KFold(n_splits=K, shuffle=True, random_state=42)

Y_res = np.empty_like(Y, dtype=float)
D_res = np.empty_like(D, dtype=float)

for train_idx, test_idx in kf.split(X):
    # 1. 只用 K-1 折训练第一阶段模型
    g_hat = RandomForestRegressor(n_estimators=200, max_depth=4, random_state=42)
    m_hat = RandomForestRegressor(n_estimators=200, max_depth=4, random_state=42)

    g_hat.fit(X[train_idx], Y[train_idx])
    m_hat.fit(X[train_idx], D[train_idx])

    # 2. 只在留出折上预测，生成折外残差
    Y_res[test_idx] = Y[test_idx] - g_hat.predict(X[test_idx])
    D_res[test_idx] = D[test_idx] - m_hat.predict(X[test_idx])

# 3. 汇总所有折外残差，估计 theta
theta_hat = np.sum(D_res * Y_res) / np.sum(D_res ** 2)

print(theta_hat)
