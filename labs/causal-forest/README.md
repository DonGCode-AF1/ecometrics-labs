# Causal Forest

This lab explains causal forests as a tool for estimating heterogeneous treatment effects.

Open the dashboard directly:

- Local file: `causal_forest_dashboard.html`
- Web preview: https://raw.githack.com/DonGCode-AF1/ecometrics-labs/master/labs/causal-forest/causal_forest_dashboard.html

## Learning Goal

The page shows how causal forests move from a single average treatment effect to conditional average treatment effects. It uses a simulated job-training example where treatment effects vary by age and skill level.

## Contents

- A visual causal tree explanation.
- A simulated dataset with observed treatment and outcome.
- CATE grouping charts.
- A practical operation flow for causal forests.
- Python code using `econml.dml.CausalForestDML`.
