因果森林：异质性处理效应的直观理解

因果森林回答的问题不是“培训平均有没有用”，而是“对谁更有用”。本页用一个职业培训案例展示 CATE 的核心思想：森林用很多棵因果树寻找相似人群，并在局部比较处理组和对照组的结果差异。

核心问题

传统平均处理效应 ATE 只给一个总体数字。因果森林估计的是条件平均处理效应：

τ(x) = E[Y(1) − Y(0) | X = x]

这里的 x 可以是年龄、教育、基线收入、技能测试分数等个体特征。τ(x) 越大，说明这类个体从处理中获得的收益越高。

随机森林预测 Y；因果森林预测“处理效应”。它关心的是在相似特征的人中，接受处理与未接受处理的结果差异。

1. 一棵因果树如何思考？

普通回归通常假设一个统一斜率；因果树会按特征切分样本，让每个叶子节点内部的人尽量相似。 在叶子节点里，它比较处理组和对照组的平均结果差：

τ̂leaf = mean(Y | W = 1, leaf) − mean(Y | W = 0, leaf)

因果森林就是很多棵这种树的平均。每棵树看一部分样本和一部分变量，因此能降低单棵树的偶然性。

4. 因果森林的操作流程

1. 准备数据Y 是结果，W 是处理，X 是所有处理前特征。
2. 残差化先估计 E[Y|X] 与 E[W|X]，减少混杂。
3. 建很多树每棵树用局部样本切分 X，寻找效应差异大的区域。
4. 叶内比较在相似人群中比较 W=1 与 W=0 的结果差异。
5. 汇总树对很多树的局部估计取平均，得到 τ̂(x)。
在观测数据中，因果森林仍依赖“条件独立/无遗漏混杂”的识别假设：给定 X 后，处理分配近似可比。算法不能自动修复没有观测到的混杂变量。

5. 有数据的代码案例

下面的代码生成与本页相同逻辑的模拟数据，并使用 Python 的 EconML 估计 CATE。若你使用 R，可以用 `grf::causal_forest` 完成同样流程。 

import numpy as np
import pandas as pd
from econml.dml import CausalForestDML
from sklearn.ensemble import RandomForestRegressor, RandomForestClassifier
from sklearn.model_selection import train_test_split

rng = np.random.default_rng(42)
n = 1200
age = rng.uniform(20, 58, n)
skill = rng.normal(0, 1, n)
baseline_income = rng.normal(8, 2, n)

X = pd.DataFrame({
    "age": age,
    "skill": skill,
    "baseline_income": baseline_income
})

# 真实异质性：年轻且技能较高者，培训收益更高
tau = 1.2 + 1.1 * (skill > 0) + 0.8 * (age < 35)

# 处理分配倾向：高技能、低收入者更可能参加培训
logit_p = -0.4 + 0.7 * skill - 0.15 * (baseline_income - 8)
p = 1 / (1 + np.exp(-logit_p))
W = rng.binomial(1, p, n)

# 结果方程：Y = 基础趋势 + W * tau(X) + noise
mu = 2.0 + 0.08 * (58 - age) + 0.7 * skill + 0.25 * baseline_income
Y = mu + W * tau + rng.normal(0, 1.2, n)

X_train, X_test, W_train, W_test, Y_train, Y_test = train_test_split(
    X, W, Y, test_size=0.3, random_state=42
)

cf = CausalForestDML(
    model_y=RandomForestRegressor(n_estimators=300, min_samples_leaf=10, random_state=1),
    model_t=RandomForestClassifier(n_estimators=300, min_samples_leaf=10, random_state=2),
    n_estimators=800,
    min_samples_leaf=20,
    random_state=42,
    discrete_treatment=True
)

cf.fit(Y_train, W_train, X=X_train)
cate_hat = cf.effect(X_test)

result = X_test.copy()
result["cate_hat"] = cate_hat
print(result.sort_values("cate_hat", ascending=False).head())
print("Estimated ATE:", cate_hat.mean())
报告结果时不要只给平均效应。更有信息量的是展示 CATE 分布、按 CATE 分组的收益、以及哪些特征推动了效应差异。
